JavaSVN 1.0 Roadmap

Legend

[-] : feature is missing
[*] : feature is in progress
[+] : feature is implemented

If the feature you're interested in is not listed below, then it probably already implemented.
To get detailed status, please contact support@tmatesoft.com.

High priority features

[-] binary properties support
[-] svn:executable property support
[-] svn:special property support (for handling versioned symlinks)
[-] javadoc documentation
[-] memory consumption optimization on large projects
[-] performance optimizations

Medium priority features

[-] diff generation support similar to "svn diff" command
[-] locking (to make JavaSVN 'process-safe')
[-] cleanup command
[-] WC->URL, URL->WC copy operations in ISVNWorkspace

Low priority features

[-] cancellation of running commands
[-] detailed progress report on running operations
[-] original svn configuration file and registry entries support
[-] ant tasks for svn operations

If you have any questions regarding JavaSVN, would like to report a bug or contribute a patch, please write to support@tmatesoft.com
